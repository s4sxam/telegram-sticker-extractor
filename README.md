# 📦 Telegram Sticker Extractor

Extract stickers from a Telegram `.zip` sticker pack and save them to your device — ready to use on Instagram!

## 🚀 Live Demo
Deploy to Vercel in one click (see below).

## ✨ Features
- Drag & drop a `.zip` sticker pack file
- Extracts all images (PNG, WebP, GIF) and videos (MP4, WebM)
- Preview all stickers in a grid
- Download individually or all at once
- Auto-converts `.webp` → `.png` for Instagram compatibility

## 📲 How to use stickers on Instagram
1. Download stickers from this tool
2. Open Instagram → Story or DM
3. Tap the 😊 sticker icon
4. Tap the **+** button
5. Pick your downloaded sticker from gallery
6. It saves to your **Recents** for quick reuse!

## 🛠 Run Locally

```bash
npm install
npm run dev
```

## ☁️ Deploy to Vercel

```bash
npm install -g vercel
vercel
```

Or connect your GitHub repo to [vercel.com](https://vercel.com) for automatic deploys.
